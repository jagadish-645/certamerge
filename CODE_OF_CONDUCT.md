# Code Of Conduct

CertaMerge contributors are expected to communicate with clarity, respect, and accountability.

The product is designed to reduce blame in software-change workflows. Contributions should follow the same standard:

- name evidence, not personal fault;
- explain policy reasons clearly;
- keep security concerns concrete;
- avoid harassment, abuse, or discriminatory language.

Maintainers may reject contributions or participation that undermines a safe and professional collaboration environment.
