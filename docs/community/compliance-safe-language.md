# Compliance-Safe Language

CertaMerge may say:

- provides machine-checkable evidence of change authorization;
- creates tamper-evident proof records for software changes;
- supports customer review, audit, and change-control workflows;
- may help reduce manual evidence collection;
- produces CARs that can be verified offline.

CertaMerge must not say:

- makes you compliant;
- guarantees security;
- certifies SOC 2, ISO 27001, SLSA, or any other standard;
- replaces an audit;
- replaces scanners;
- proves code is safe.

Required disclaimer:

```text
CertaMerge provides evidence that may support compliance programs. It does not certify or guarantee compliance with any standard.
```
