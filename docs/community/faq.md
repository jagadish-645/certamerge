# FAQ

## Is CertaMerge an AI code reviewer?

No. It does not judge code quality. It evaluates proof and policy.

## Is CertaMerge a scanner?

No. It consumes scanner evidence where available. It does not replace CodeQL, Semgrep, Trivy, Gitleaks, or dependency scanners.

## Does CertaMerge make code secure?

No. It provides evidence that may support review and change-control decisions.

## Does CertaMerge certify compliance?

No. It may provide change authorization evidence that supports audit or security review workflows.

## Does CertaMerge send source code anywhere?

No, not by default. Community commands run locally and emit metadata, evidence states, missing proof, repair missions, and CARs.
