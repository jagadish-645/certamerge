# Community Feedback

CertaMerge community alpha feedback should stay tied to proof-carrying workflow behavior.

Useful feedback includes:

- whether Recover found a real proof gap;
- whether Gate produced a verdict a reviewer would respect;
- whether the missing proof was concrete;
- whether the accountable next action was clear;
- whether the CAR was verifiable and useful later.

Do not share secrets, tokens, private keys, raw proprietary source code, raw restricted diffs, customer production data, or private organization names.

Use the issue templates for bugs, alpha feedback, proof-gap examples, and evidence adapter requests.
